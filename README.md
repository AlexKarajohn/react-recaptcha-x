# react-recaptcha-v3-v2

a React reCAPTCHA version 3 with version 2 fallback component

# development environment

## prerequisites

1. [nodejs](https://nodejs.org/en/) 10.15.x LTS

## setup

1. `cd src/module`
1. `npm install`
1. `cd ../dev`
1. `npm install`
1. `npm run`
1. open browser to `http://locahost:9000`
